# Smart Camera Object Recognition App 📷

A **Smart Camera Object Recognition Application** that detects objects using a camera and processes them in real time. The system captures input from a camera, analyzes the visual data, and identifies objects to demonstrate practical computer vision applications.

This project was built as part of a **hackathon**, focusing on creating an intelligent camera system capable of recognizing objects in its environment.

---

## 🚀 Features

* 📷 Camera-based object detection
* ⚡ Real-time object recognition
* 🧠 Smart processing of visual input
* 🖥️ Displays detected objects from the camera feed

---

## 🧰 Tech Stack

* **JavaScript**
* **React.js**
* **Node.js**
* **Express.js**
* **Gemini AI API**
* **Computer Vision / Object Recognition APIs**

---

## ⚙️ How It Works

1. The application captures input from a connected camera.
2. The captured frames are processed by the recognition system.
3. The system analyzes the objects present in the frame.
4. Detected objects are identified and displayed.

## 🛠 Installation

### Clone the repository

```bash
git clone https://github.com/Shiva-Prasad83/Smart-Camera-Object-Recognition-App-
cd Smart-Camera-Object-Recognition-App-
```

### Install dependencies

```bash
npm install
```

### Run the application

```bash
npm start
```
